# Developer Notes

This folder is for active development notes, implementation plans, debugging workflows, and project-specific experiments that are useful to keep with the repository but are not part of the main user-facing setup docs.

Use this folder for notes such as:

- Quest runtime debugging workflows.
- Unity debug-panel plans.
- ROS/Unity integration experiments.
- Design decisions and tradeoffs.
- Debug logs summaries and reproduction notes.
- Future feature plans.

Do not put large recordings, datasets, APKs, screenshots, or temporary raw logs here. Store those outside Git or in external storage, and link them from a note if needed.

Recommended naming:

```text
YYYY_MM_DD_short_topic.md
Unity_Quest_Debug_Stream_Setup.md
Runtime_Debug_Panel_Plan.md
Input_Replay_Design.md
```

If a note becomes stable and useful for other users, move or summarize it into `docs/System_Setup.md`, `docs/Getting_Started.md`, or `docs/Technical_Details.md`.
